## 本阶段任务

本目录**不设** `pc/` 练习代码。序曲用于建立整体图景，再进入 ch01 手写。

1. 读完序曲章节（约 15–30 分钟）。
2. 运行官方教学包预热（不必手写）：

```bash
cd oop-in-c/code/01-three-leds/pc
make
./demo
```

3. 在 [`LEARNING_PLAN.md`](../LEARNING_PLAN.md) 总进度表勾选「序曲」。
4. 进入 [`01-封装/ch01-three-leds/pc/`](../01-封装/ch01-three-leds/pc/) 开始第一章练习。

## 关注点

- `struct` + 指针 ≈ 面向对象中的「对象」
- 换引脚只改 init 参数，应用 API 不变
- 为后续封装 / 继承 / 多态章节建立直觉
